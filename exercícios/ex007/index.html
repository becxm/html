Texto grande e pequeno.

Para texto grande apenas com CSS, mas seria a tag big.

Para texto pequeno usamos a tag small.

Texto deletado e inserido.

Para deletar textos usamos a tag del.

Para inserir textos usamos a tag ins.

Para apenas sublinhar usamos a tag U (não-semântica).

Texto sobrescrito e subscrito.

Para sobrescrever coisas do tipo x20 usamos a tag sup.

Para subscrever coisas do tipo H2O usamos a tag sub.
